# SUN Education Website

A responsive educational institute website developed using HTML, CSS and Bootstrap.

## Features

- Responsive navigation menu
- Multi-level dropdown menus
- Hero section
- About section
- Courses section
- Director's message
- Principal's message
- Contact form
- Mobile-friendly layout

## Technologies Used

- HTML5
- CSS3
- Bootstrap
- Font Awesome

## Project Purpose

This project was developed as a frontend web development practice project during my engineering studies.
